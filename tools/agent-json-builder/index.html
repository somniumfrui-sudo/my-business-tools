⚙️ AIエージェント用・構造化指示ジェネレーター

自然文で「やりたいこと」を書くだけで、AIエージェントが確実に読み取れるJSON指示書を自動生成します。

🖊 指示を入力
テンプレートから始める
0.3
📄 生成されたJSON指示書
{ } 左のフォームを入力して「生成」を押してください
🚀 使い方ガイド
生成されたJSONは以下のフレームワークで直接使用できます：
LangChain AutoGen CrewAI Claude MCP Dify n8n
💡 「プロンプト」タブに切り替えると、そのままChatGPT・Claudeに貼り付けて使えるテキスト形式で出力されます。
フィールド定義 — AgentInstruction v1.0 スキーマ
agent_id
string
エージェントの一意識別子
version
string
指示書のバージョン管理
role
string
エージェントの役割・専門性
goal
string
達成すべき最終目標
context
object
実行環境・背景情報
tools
array
使用可能なツール一覧
constraints
array
禁止事項・制約ルール
output_spec
object
出力形式の詳細仕様
execution
object
実行パラメータ（ループ数・温度等）
